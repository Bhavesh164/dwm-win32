# dwm-win32 lua api

# dwm module

```lua
local dwm = require 'dwm'
```

## dwm.log

```lua
local dwm = require 'dwm'
dwm.log('Hello World!')
```

## dwm.VERSION

```lua
local dwm = require 'dwm'
dwm.log(dwm.VERSION) -- ex: 1.0.0
```

## dwm.PLATFORM

```lua
local dwm = require 'dwm'
dwm.log(dwm.PLATFORM) -- ex: Windows
```
